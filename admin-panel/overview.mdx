# Admin Panel Overview

The Gridix admin panel is designed for real estate developers to efficiently manage projects, leads, team members, and integrations. This guide provides a comprehensive overview of all available features.

## Accessing the Admin Panel

After logging in to your Gridix account, you'll be automatically redirected to the admin dashboard. The dashboard is accessible from any device with an internet connection.

<Warning>
Ensure you have a stable internet connection for optimal performance. Some features may require JavaScript to be enabled in your browser.
</Warning>

## Main Interface Components

### Sidebar Navigation

The left sidebar contains all main sections:

- **Projects** - Manage your real estate projects
- **Leads** - Track and manage incoming leads
- **Subscription** - View and manage your subscription plan
- **Partners** - Access partner program features
- **Widgets** - Generate embed codes
- **Analytics** - View project and lead statistics (coming soon)
- **Settings** - Configure account and company settings

### Content Area

The main content area displays the selected section. It adapts to show:
- Project lists and details
- Lead tables with filters
- Settings forms
- Widget configuration
- And more

### Top Bar

The top bar includes:
- **Logo/Branding** - Your company branding
- **Language Toggle** - Switch between supported languages
- **User Menu** - Profile and logout options
- **Support Button** - Quick access to help

## Dashboard Features

### Project Management

<Card title="Project List" icon="list">
View all your projects in one place with:
- Project cards showing key information
- Quick edit access
- Status indicators
- Search and filter capabilities
</Card>

From the Projects tab, you can:
- Create new projects (manual or import)
- Edit existing projects
- View project statistics
- Delete projects (with confirmation)

### Lead Management

<Card title="Leads Dashboard" icon="users">
Comprehensive lead tracking with:
- Real-time lead statistics
- Advanced filtering options
- Status management
- CRM integration status
</Card>

The Leads tab provides:
- Overview of all leads across projects
- Filter by project, status, or date
- Add notes to leads
- Export lead data
- View CRM sync status

### Team Management

<Card title="Manager Accounts" icon="user-check">
Manage team members with:
- Role-based access control
- Workspace assignments
- Project-level permissions
</Card>

Access team management from Settings → Manager Accounts:
- Add new managers
- Assign to workspaces
- Configure project access
- Remove team members

### Widget Configuration

<Card title="Widget Generator" icon="code">
Generate embed codes with:
- Project selection
- Language configuration
- Preview functionality
- Copy-ready HTML code
</Card>

## User Roles

Gridix supports different user roles:

<Tabs>
  <Tab title="Developer">
    Full access to all features:
    - Create and manage projects
    - Access all leads
    - Manage team members
    - Configure integrations
    - Manage subscriptions
  </Tab>
  
  <Tab title="Manager">
    Limited access based on workspace assignment:
    - View assigned projects
    - Access leads from assigned projects
    - Limited settings access
    - Cannot manage subscriptions
  </Tab>
</Tabs>

## Customization Options

### Theme Customization

Customize the admin panel appearance:
- Brand colors
- Logo settings
- Theme preferences

<Note>
Theme customization is available in Settings → Branding (for developers only).
</Note>

### Language Settings

Switch between supported languages:
- English (en)
- Russian (ru)
- Georgian (ka)
- Arabic (ar)

Language preference is saved per user and persists across sessions.

## Keyboard Shortcuts

<Tip>
Use these keyboard shortcuts for faster navigation:
- `Ctrl/Cmd + K` - Open search
- `Esc` - Close modals
- `Tab` - Navigate between fields
- `Enter` - Submit forms
</Tip>
