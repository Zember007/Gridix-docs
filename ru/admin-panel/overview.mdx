# Обзор панели администратора

Админ-панель Gridix разработана для застройщиков недвижимости, чтобы эффективно управлять проектами, лидами, членами команды и интеграциями. Это руководство предоставляет полный обзор всех доступных функций.

## Доступ к админ-панели

После входа в ваш аккаунт Gridix вы будете автоматически перенаправлены в админ-панель. Панель управления доступна с любого устройства с подключением к интернету.

<Warning>
Убедитесь, что у вас стабильное интернет-соединение для оптимальной производительности. Некоторые функции могут требовать включения JavaScript в вашем браузере.
</Warning>

## Основные компоненты интерфейса

### Боковая панель навигации

Левая боковая панель содержит все основные разделы:

- **Проекты** - Управляйте вашими проектами недвижимости
- **Лиды** - Отслеживайте и управляйте входящими лидами
- **Подписка** - Просматривайте и управляйте вашим планом подписки
- **Партнеры** - Доступ к функциям партнерской программы
- **Виджеты** - Генерируйте коды для встраивания
- **Аналитика** - Просматривайте статистику проектов и лидов (скоро)
- **Настройки** - Настройте аккаунт и настройки компании

### Область контента

Основная область контента отображает выбранный раздел. Она адаптируется для отображения:
- Списков проектов и деталей
- Таблиц лидов с фильтрами
- Форм настроек
- Конфигурации виджетов
- И многого другого

### Верхняя панель

Верхняя панель включает:
- **Логотип/Брендинг** - Брендинг вашей компании
- **Переключение языка** - Переключение между поддерживаемыми языками
- **Меню пользователя** - Профиль и опции выхода
- **Кнопка поддержки** - Быстрый доступ к помощи

## Функции панели управления

### Управление проектами

<Card title="Список проектов" icon="list">
Просматривайте все ваши проекты в одном месте с:
- Карточками проектов, показывающими ключевую информацию
- Быстрым доступом к редактированию
- Индикаторами статуса
- Возможностями поиска и фильтрации
</Card>

Из вкладки Проекты вы можете:
- Создавать новые проекты (вручную или импорт)
- Редактировать существующие проекты
- Просматривать статистику проектов
- Удалять проекты (с подтверждением)

### Управление лидами

<Card title="Панель лидов" icon="users">
Комплексное отслеживание лидов с:
- Статистикой лидов в реальном времени
- Расширенными опциями фильтрации
- Управлением статусом
- Статусом интеграции CRM
</Card>

Вкладка Лиды предоставляет:
- Обзор всех лидов по проектам
- Фильтрацию по проекту, статусу или дате
- Добавление заметок к лидам
- Экспорт данных лидов
- Просмотр статуса синхронизации CRM

### Управление командой

<Card title="Аккаунты менеджеров" icon="user-check">
Управляйте членами команды с:
- Контролем доступа на основе ролей
- Назначениями рабочих пространств
- Разрешениями на уровне проекта
</Card>

Доступ к управлению командой из Настройки → Аккаунты менеджеров:
- Добавляйте новых менеджеров
- Назначайте на рабочие пространства
- Настраивайте доступ к проектам
- Удаляйте членов команды

### Конфигурация виджетов

<Card title="Генератор виджетов" icon="code">
Генерируйте коды для встраивания с:
- Выбором проекта
- Конфигурацией языка
- Функцией предварительного просмотра
- Готовым HTML кодом для копирования
</Card>

## Роли пользователей

Gridix поддерживает различные роли пользователей:

<Tabs>
  <Tab title="Разработчик">
    Полный доступ ко всем функциям:
    - Создавать и управлять проектами
    - Доступ ко всем лидам
    - Управлять членами команды
    - Настраивать интеграции
    - Управлять подписками
  </Tab>
  
  <Tab title="Менеджер">
    Ограниченный доступ на основе назначения рабочего пространства:
    - Просматривать назначенные проекты
    - Доступ к лидам из назначенных проектов
    - Ограниченный доступ к настройкам
    - Не может управлять подписками
  </Tab>
</Tabs>

## Опции настройки

### Настройка темы

Настройте внешний вид админ-панели:
- Цвета бренда
- Настройки логотипа
- Предпочтения темы

<Note>
Настройка темы доступна в Настройки → Брендинг (только для разработчиков).
</Note>

### Языковые настройки

Переключение между поддерживаемыми языками:
- Английский (en)
- Русский (ru)
- Грузинский (ka)
- Арабский (ar)

Предпочтение языка сохраняется для каждого пользователя и сохраняется между сессиями.

## Мобильный доступ

<Check>
Админ-панель полностью адаптивна и оптимизирована для:
- Настольных браузеров (Chrome, Firefox, Safari, Edge)
- Планшетных устройств (iPad, планшеты Android)
- Мобильных телефонов (iOS, Android)
</Check>

Мобильный интерфейс включает:
- Сворачиваемое боковое меню
- Оптимизированные для касания элементы управления
- Адаптивные таблицы и карточки
- Оптимизированные формы

## Горячие клавиши

<Tip>
Используйте эти горячие клавиши для более быстрой навигации:
- `Ctrl/Cmd + K` - Открыть поиск
- `Esc` - Закрыть модальные окна
- `Tab` - Навигация между полями
- `Enter` - Отправить формы
</Tip>
