# Обзор панели управления

Админ-панель Gridix - это ваш центральный узел для управления проектами, лидами, настройками и многим другим. Это руководство поможет вам понять интерфейс и эффективно навигироваться.

## Главная навигация

Админ-панель состоит из нескольких основных разделов, доступных из боковой панели:

<CardGroup cols={2}>
  <Card title="Проекты" icon="building">

  </Card>
  <Card title="Лиды" icon="users">
    Отслеживайте и управляйте лидами из ваших проектов. Фильтруйте по статусу, проекту или диапазону дат.
  </Card>
  <Card title="Виджеты" icon="code">
    Генерируйте коды для встраивания ваших проектов. Настройте параметры виджета и предварительный просмотр перед встраиванием.
  </Card>
  <Card title="Настройки" icon="settings">

## Разделы панели управления

### Вкладка Проекты

Вкладка Проекты отображает все ваши проекты в виде карточек. Отсюда вы можете:

- **Создать новый проект**: Нажмите кнопку "+", чтобы начать новый проект
- **Просмотреть проекты**: Увидите все ваши проекты с их статусом, количеством квартир и последним обновлением
- **Редактировать проекты**: Нажмите на любой проект, чтобы открыть редактор проекта
- **Фильтровать и искать**: Используйте фильтры для быстрого поиска конкретных проектов

<Note>
Проекты перечислены с наиболее недавно обновленными первыми. Используйте строку поиска, чтобы найти проекты по имени.
</Note>

### Вкладка Лиды

Вкладка Лиды предоставляет комплексное управление лидами:

- **Просмотреть все лиды**: Увидите все лиды из всех проектов в одном месте
- **Панель статистики**: Просматривайте количество по статусу (ожидающие, отправленные в CRM, только сохраненные, неудачные, отмененные)
- **Фильтровать лиды**: Фильтруйте по проекту, статусу или диапазону дат
- **Действия с лидами**: Добавляйте заметки, отменяйте лиды, копируйте информацию о лидах или открывайте в CRM

<Info>
Лиды автоматически создаются, когда посетители отправляют запросы через ваши встроенные виджеты.
</Info>

### Вкладка Виджеты

Вкладка Виджеты позволяет вам:

- **Генерировать код для встраивания**: Создавайте код виджета для встраивания на ваш веб-сайт
- **Выбрать проект**: Выберите, какой проект встроить (или все проекты)
- **Настроить язык**: Установите язык по умолчанию для виджета
- **Предварительный просмотр виджета**: Посмотрите, как будет выглядеть виджет перед встраиванием
- **Копировать код**: Получите готовый HTML код для вашего веб-сайта

### Вкладка Настройки

Вкладка Настройки включает:

- **Профиль компании**: Обновите название компании, контактную информацию и бизнес-детали
- **Аккаунты менеджеров**: Добавляйте и управляйте членами команды с доступом на основе ролей
- **Подписка**: Просматривайте и управляйте вашим планом подписки
- **Настройка темы**: Настройте цвета бренда и визуальные параметры

### Вкладка Партнеры

Вкладка Партнеры предоставляет:

- **Обзор партнерской программы**: Информация о том, как стать партнером
- **Реферальные ссылки**: Генерируйте и делитесь партнерскими реферальными ссылками
- **Управление клиентами**: Управляйте клиентами и отслеживайте заработок
- **Статистика**: Просматривайте показатели производительности партнера

### Вкладка Подписка

Вкладка Подписка показывает:

- **Текущий план**: Просмотрите вашу активную подписку и функции
- **Варианты планов**: Просмотрите доступные планы (Basic и Pro)
- **История платежей**: Доступ к счетам и истории платежей
- **Обновление/Понижение**: Измените ваш план по мере необходимости

## Ключевые функции

### Быстрые действия

По всей панели управления вы найдете кнопки быстрых действий:

- **Кнопка помощи**: Фиксированная кнопка в правом нижнем углу, ведущая к поддержке Telegram
- **Переключение языка**: Переключение между поддерживаемыми языками (английский, русский, грузинский, арабский)
- **Меню пользователя**: Доступ к вашему профилю и выходу

### Адаптивность для мобильных устройств

<Check>
Админ-панель полностью адаптивна и работает на:
- Настольных компьютерах
- Планшетах
- Мобильных телефонах
</Check>

Мобильный интерфейс автоматически адаптируется с гамбургер-меню для навигации.

## Советы по эффективной навигации

<Tip>
Используйте горячие клавиши, где доступно:
- Нажмите `/`, чтобы сфокусироваться на строке поиска
- Используйте `Tab` для навигации между разделами
- `Esc` для закрытия модальных окон и диалогов
</Tip>

<Tip>
Сохраняйте часто используемые страницы в закладках браузера для быстрого доступа к конкретным проектам или разделам.
</Tip>
