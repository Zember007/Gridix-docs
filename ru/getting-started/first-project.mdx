# Создайте ваш первый проект

Это руководство поможет вам создать ваш первый проект в Gridix. Вы можете создавать проекты, импортируя данные из Excel/Google Sheets или настраивая их вручную.

## Предварительные требования

Перед началом убедитесь, что у вас есть:
- Подтвержденный аккаунт Gridix
- Доступ к админ-панели
- Данные проекта готовы (номера квартир, этажи, комнаты, площадь, цена, статус)

## Выбор метода создания

Gridix предлагает два способа создания проекта:

<Tabs>
  <Tab title="Импорт из Excel/Google Sheets">
    Лучше всего для проектов с существующими данными в таблицах. Поддерживает:
    - Файлы Excel (.xlsx, .xls)
    - CSV файлы
    - Google Sheets (через ссылку)
    
    <Note>
    Импорт по ссылке позволяет автоматическую синхронизацию при обновлении вашей таблицы.
    </Note>
  </Tab>
  
  <Tab title="Ручная настройка">
    Лучше всего для проектов, которые вы хотите настроить с нуля. Дает вам полный контроль над:
    - Структурой проекта
    - Конфигурацией полей
    - Настройками визуализации
    - Пользовательскими свойствами
  </Tab>
</Tabs>

## Метод 1: Импорт из Excel

<Steps>
  <Step title="Откройте создание проекта">
    В админ-панели нажмите "Проекты" в боковой панели, затем нажмите "Создать новый проект".
  </Step>
  
  <Step title="Выберите метод импорта">
    Выберите вкладку "Импорт из Excel". Вы можете:
    - Загрузить файл с вашего компьютера
    - Импортировать из Google Sheets, используя ссылку
  </Step>
  
  <Step title="Подготовьте ваши данные">
    Убедитесь, что ваш файл Excel содержит необходимые столбцы:
    - Номер квартиры
    - Номер этажа
    - Количество комнат
    - Площадь (в квадратных метрах)
    - Цена
    - Статус (доступна, продана, забронирована и т.д.)
    
    <Tip>
    Скачайте наш шаблонный файл, чтобы обеспечить правильное форматирование: Нажмите "Скачать шаблон" в диалоге импорта.
    </Tip>
  </Step>
  
  <Step title="Загрузите или свяжите файл">
    - **Загрузка файла**: Нажмите "Загрузить файл Excel" и выберите ваш файл
    - **Google Sheets**: Вставьте ссылку на Google Sheets (поддерживается любой формат)
  </Step>
  
  <Step title="Сопоставьте столбцы">
    Gridix автоматически определит ваши столбцы. Просмотрите и при необходимости отрегулируйте сопоставление:
    - Сопоставьте названия ваших столбцов с полями Gridix
    - Проверьте правильность типов данных
    - Добавьте любые пользовательские поля, которые вам нужны
  </Step>
  
  <Step title="Настройте планировку этажей">
    Ответьте на вопрос о планировке этажей:
    - **Одинаковая планировка на всех этажах**: Отметьте квартиры один раз и примените ко всем этажам
    - **Разные планировки**: Настройте каждый этаж отдельно
    
    <Info>
    Если планировки одинаковые, вы сэкономите значительное время при настройке.
    </Info>
  </Step>
  
  <Step title="Просмотрите и создайте">
    Просмотрите предварительный просмотр импортированных данных, затем нажмите "Создать проект" для завершения.
  </Step>
</Steps>

## Метод 2: Ручная настройка

<Steps>
  <Step title="Начните ручное создание">
    В диалоге создания проекта выберите "Ручная настройка" и нажмите "Начать ручное создание".
  </Step>
  
  <Step title="Введите детали проекта">
    Заполните основную информацию о проекте:
    - **Название проекта**: Название вашей застройки
    - **Описание**: Краткое описание проекта
    - **Адрес**: Физический адрес здания
    - **Координаты**: Широта и долгота (необязательно, для отображения на карте)
    - **Изображение здания**: Загрузите основное изображение здания
  </Step>
  
  <Step title="Настройте параметры здания">
    Установите свойства уровня здания:
    - Количество этажей
    - Валюта для ценообразования
    - Тип проекта (здание, комплекс и т.д.)
    - Наличие парковки
    - Наличие коммерческих помещений
  </Step>
  
  <Step title="Загрузите планы этажей">
    Загрузите изображения планов этажей для каждого этажа:
    - Нажмите "Загрузить план этажа" для каждого этажа
    - Поддерживаемые форматы: PNG, JPG, SVG
    - Рекомендуемый размер: 1920x1080 или выше
  </Step>
  
  <Step title="Отметьте квартиры на планах этажей">
    Для каждого плана этажа:
    - Нажмите, чтобы отметить расположение квартир
    - Введите номера квартир
    - Настройте свойства квартир (комнаты, площадь, цена, статус)
  </Step>
  
  <Step title="Сохраните проект">
    Нажмите "Сохранить", чтобы создать ваш проект. Вы можете продолжить редактирование после создания.
  </Step>
</Steps>

## После создания проекта

<Check>
После создания вашего проекта вы можете:
- Редактировать детали и настройки проекта
- Загружать дополнительные планы этажей
- Настраивать свойства квартир
- Настроить ценообразование и калькулятор рассрочки
- Настроить параметры визуализации
- Генерировать код виджета для встраивания
</Check>
