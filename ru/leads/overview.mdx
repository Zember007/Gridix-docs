# Обзор лидов

Gridix автоматически собирает лиды, когда посетители взаимодействуют с вашими встроенными планами этажей. Это руководство объясняет, как работает система лидов и какая информация собирается.

## Что такое лиды?

Лид — это потенциальный клиент, который проявил интерес к вашим квартирам, выполнив одно из действий:
- Отправил форму запроса
- Запросил дополнительную информацию
- Нажал "Связаться" на квартире
- Забронировал просмотр

<Check>
Каждый лид содержит:
- Имя клиента
- Email адрес
- Номер телефона
- Квартиру интереса
- Информацию о проекте
- Временную метку отправки
</Check>

## Как создаются лиды

### Автоматический сбор

Лид автоматически создается, когда:

<Steps>
  <Step title="Взаимодействие клиента">
    Посетитель взаимодействует с вашим встроенным виджетом плана этажа.
  </Step>
  
  <Step title="Отправка формы">
    Клиент заполняет и отправляет форму запроса на квартиру.
  </Step>
  
  <Step title="Создание лида">
    Gridix автоматически создает лид со всей отправленной информацией.
  </Step>
  
  <Step title="Уведомление">
    Вы получаете уведомление (если включено) о новом лиде.
  </Step>
</Steps>

### Информация о лиде

Каждый лид включает:

<CardGroup cols={2}>
  <Card title="Детали клиента">
    - Полное имя
    - Email адрес
    - Номер телефона
    - Предпочитаемый способ связи
  </Card>
  <Card title="Информация о недвижимости">
    - Название проекта
    - Номер квартиры
    - Детали квартиры (комнаты, площадь, цена)
    - Номер этажа
  </Card>
</CardGroup>

## Статус лида

### Типы статусов

Лид может иметь различные статусы:

- **Ожидает**: Недавно создан, ожидает обработки
- **Отправлено в CRM**: Успешно отправлено в интегрированную CRM-систему
- **Только сохранено**: Сохранено в Gridix, но не отправлено в CRM
- **Неудачно**: Не удалось отправить в CRM (будет повторная попытка)
- **Отменено**: Лид отменен или недействителен

### Поток статусов

```
Ожидает → Отправлено в CRM → (Обработка CRM)
   ↓
Только сохранено (если CRM не настроена)
   ↓
Неудачно (если ошибка CRM) → Повторная попытка → Отправлено в CRM
```

## Источники лидов

Лид может поступать из различных источников:

- **Виджет сайта**: Встроенный план этажа на вашем сайте
- **Прямая ссылка**: Общая ссылка проекта
- **Собственный домен**: Лиды с собственного домена
- **Мобильное приложение**: Если используется мобильное приложение

<Info>
Все лиды отслеживаются с их источником, помогая вам понять, какие каналы наиболее эффективны.
</Info>

## Статистика лидов

### Обзор панели

Вкладка Лиды показывает:

- **Всего лидов**: Все лиды по всем проектам
- **Ожидает**: Лиды, ожидающие обработки
- **Отправлено в CRM**: Успешно интегрированные лиды
- **Только сохранено**: Лиды сохранены, но не отправлены
- **Неудачно**: Лиды с ошибками доставки
- **Отменено**: Отмененные лиды

### Статистика по проектам

Просмотрите статистику по проектам:
- Лиды по проектам
- Коэффициенты конверсии
- Наиболее популярные квартиры
- Разбивка источников лидов

## Функции управления лидами

### Просмотр лидов

Доступ к лидам из:
- **Вкладка Лиды**: Все лиды в одном месте
- **Представление проекта**: Лиды, отфильтрованные по проекту
- **Детали лида**: Информация об отдельном лиде

### Фильтрация лидов

Фильтруйте лиды по:
- **Проект**: Показать лиды из конкретных проектов
- **Статус**: Фильтр по статусу лида
- **Диапазон дат**: Лиды за определенный период времени
- **Квартира**: Лиды для конкретных квартир
- **Источник**: Фильтр по источнику лида

### Действия с лидами

Доступные действия с лидами:

- **Просмотр деталей**: Увидеть полную информацию о лиде
- **Добавить заметки**: Внутренние заметки для команды
- **Обновить статус**: Изменить статус лида вручную
- **Копировать информацию**: Скопировать детали лида
- **Открыть в CRM**: Прямая ссылка в CRM (если интегрирована)
- **Отменить лид**: Отметить лид как отмененный

## Интеграция CRM

### Автоматическая отправка в CRM

Когда CRM настроена:

<Steps>
  <Step title="Лид создан">
    Новый лид создается в Gridix.
  </Step>
  
  <Step title="Отправка в CRM">
    Лид автоматически отправляется в вашу CRM-систему.
  </Step>
  
  <Step title="Обновление статуса">
    Статус лида обновляется на "Отправлено в CRM" при успехе.
  </Step>
  
  <Step title="Обработка CRM">
    Лид появляется в вашей CRM для последующих действий.
  </Step>
</Steps>

### Ручная синхронизация CRM

Для лидов, которые не удались или были только сохранены:
- Вручную повторите отправку в CRM
- Экспортируйте и импортируйте в CRM
- Скопируйте информацию о лиде для ручного ввода
