# Создание проектов

Проекты - это основа Gridix. Каждый проект представляет собой застройку недвижимости с планами этажей, квартирами и ценообразованием. Это руководство охватывает как ручное создание, так и методы импорта.

## Обзор методов создания

Gridix предлагает два основных метода создания проектов:

<CardGroup cols={2}>
  <Card title="Импорт из Excel/Google Sheets" icon="file-spreadsheet">
    Быстро и эффективно для проектов с существующими данными. Поддерживает файлы Excel и ссылки Google Sheets с автоматической синхронизацией.
  </Card>
  <Card title="Ручная настройка" icon="settings">
    Полный контроль над структурой проекта. Идеально для новых застроек или когда вам нужна полная настройка.
  </Card>
</CardGroup>

## Метод 1: Импорт из Excel

Импорт из Excel - это самый быстрый способ создать проект, когда у вас уже есть данные в таблице.

### Подготовка ваших данных

Ваш файл Excel должен содержать следующие столбцы:

| Столбец | Описание | Обязательно | Пример |
|---------|----------|-------------|--------|
| Номер квартиры | Уникальный идентификатор квартиры | ✅ | 101, 102, A-1 |
| Этаж | Номер этажа | ✅ | 1, 2, 3 |
| Комнаты | Количество комнат | ✅ | 1, 2, 3, 4+ |
| Площадь | Площадь в квадратных метрах | ✅ | 45.5, 68.2 |
| Цена | Цена квартиры | ✅ | 5500000, 7200000 |
| Статус | Статус доступности | ✅ | свободна, продана, забронирована |

<Tip>
Скачайте шаблонный файл из диалога импорта, чтобы обеспечить правильное форматирование. Шаблон включает все необходимые столбцы с примерами данных.
</Tip>

### Процесс импорта

<Steps>
  <Step title="Откройте создание проекта">
    В админ-панели перейдите в Проекты и нажмите "Создать новый проект".
  </Step>
  
  <Step title="Выберите метод импорта">
    Выберите вкладку "Импорт из Excel". Вы увидите две опции:
    - **Загрузка файла**: Загрузите файл Excel с вашего компьютера
    - **Импорт по ссылке**: Подключитесь к Google Sheets через ссылку
  </Step>
  
  <Step title="Загрузите или свяжите файл">
    **Для загрузки файла:**
    - Нажмите "Загрузить файл Excel"
    - Выберите ваш файл .xlsx, .xls или .csv
    - Дождитесь обработки файла
    
    **Для Google Sheets:**
    - Нажмите вкладку "По ссылке"
    - Вставьте ссылку на Google Sheets (поддерживается любой формат)
    - Gridix автоматически преобразует и синхронизирует
  </Step>
  
  <Step title="Сопоставьте столбцы">
    Gridix автоматически определяет столбцы, но вы можете настроить:
    - Просмотрите обнаруженные сопоставления столбцов
    - Сопоставьте названия ваших столбцов с полями Gridix
    - Проверьте типы данных
    - Добавьте пользовательские поля, если нужно
  </Step>
  
  <Step title="Настройте планировку этажей">
    Ответьте на вопрос о планировке этажей:
    - **Одинаковая планировка**: Отметьте квартиры один раз, примените ко всем этажам
    - **Разные планировки**: Настройте каждый этаж отдельно
    
    <Info>
    Если все этажи имеют одинаковую планировку, вы сэкономите значительное время при настройке.
    </Info>
  </Step>
  
  <Step title="Просмотрите данные">
    Просмотрите предварительный просмотр импортированных данных:
    - Проверьте количество квартир
    - Проверьте точность данных
    - Просмотрите любые предупреждения или ошибки
  </Step>
  
  <Step title="Создайте проект">
    Нажмите "Создать проект" для завершения. Проект будет создан со всеми квартирами и данными.
  </Step>
</Steps>

### Преимущества импорта по ссылке

<CardGroup cols={3}>
  <Card title="Всегда актуально" icon="refresh">
    Данные автоматически синхронизируются при обновлении вашей таблицы. Нет необходимости повторно загружать файлы.
  </Card>
  <Card title="Не нужно повторно загружать" icon="upload">

  <Card title="Синхронизация в реальном времени" icon="clock">

  </Card>
</CardGroup>

## Метод 2: Ручная настройка

Ручная настройка дает вам полный контроль над конфигурацией проекта.

### Начало ручного создания

<Steps>
  <Step title="Откройте создание проекта">
    Перейдите в Проекты → "Создать новый проект".
  </Step>
  
  <Step title="Выберите ручную настройку">
    Выберите вкладку "Ручная настройка" и нажмите "Начать ручное создание".
  </Step>
  
  <Step title="Введите основную информацию">
    Заполните детали проекта:
    - **Название проекта**: Название вашей застройки
    - **Описание**: Краткое описание (необязательно)
    - **Адрес**: Физический адрес
    - **Координаты**: Широта и долгота (для отображения на карте)
    - **Изображение здания**: Загрузите основное изображение здания
  </Step>
  
  <Step title="Настройте параметры здания">
    Установите свойства здания:
    - Количество этажей
    - Валюта (USD, EUR, GEL, RUB и т.д.)
    - Тип проекта (здание, комплекс и т.д.)
    - Наличие парковки
    - Наличие коммерческих помещений
    - Настройки калькулятора рассрочки
  </Step>
  
  <Step title="Сохраните основную информацию">
    Нажмите "Сохранить" для создания проекта. Вы можете продолжить добавление деталей.
  </Step>
</Steps>

### Добавление планов этажей

<Steps>
  <Step title="Перейдите к планам этажей">
    В редакторе проекта перейдите в раздел "Планы этажей".
  </Step>
  
  <Step title="Загрузите план этажа">
    Для каждого этажа:
    - Нажмите "Загрузить план этажа"
    - Выберите файл изображения (PNG, JPG, SVG)
    - Рекомендуемый размер: 1920x1080 или выше
    - Добавьте номер этажа
  </Step>
  
  <Step title="Отметьте квартиры">
    На загруженном плане этажа:
    - Нажмите, чтобы отметить расположение квартир
    - Введите номер квартиры
    - Установите начальные свойства (комнаты, площадь, цена)
  </Step>
  
  <Step title="Повторите для всех этажей">
    Загрузите и настройте планы этажей для каждого этажа в здании.
  </Step>
</Steps>

## Опции конфигурации проекта

### Основные настройки

- **Название проекта**: Отображаемое имя проекта
- **Slug**: URL-дружественный идентификатор (автоматически генерируется)
- **Описание**: Подробное описание проекта
- **Адрес**: Полный адрес с городом и страной
- **Координаты**: GPS координаты для интеграции карты

### Свойства здания

- **Этажи**: Общее количество этажей
- **Валюта**: Валюта ценообразования
- **Тип проекта**: Классификация типа здания
- **Парковка**: Наличие парковки (да/нет)
- **Коммерческие помещения**: Наличие коммерческих помещений
- **Вид фасада**: Включить 3D визуализацию фасада

### Настройки ценообразования

- **Валюта по умолчанию**: Базовая валюта для всех цен
- **Калькулятор рассрочки**: Включить/выключить расчеты рассрочки
- **Минимальный первоначальный взнос**: Минимальный процент первоначального взноса
- **Максимальные месяцы рассрочки**: Максимальный период рассрочки

## Поддерживаемые форматы файлов

### Импорт Excel

- **Excel**: .xlsx, .xls
- **CSV**: .csv
- **Google Sheets**: Любой формат доступной ссылки

### Изображения планов этажей

- **PNG**: Рекомендуется для детальных планов этажей
- **JPG**: Хорошо для фотографий
- **SVG**: Векторный формат для масштабируемости

## Устранение неполадок

<AccordionGroup>
  <Accordion title="Импорт не удался">
    - Проверьте формат файла (поддерживаются Excel, CSV)
    - Убедитесь, что присутствуют необходимые столбцы
    - Убедитесь, что данные в правильном формате
    - Проверьте размер файла (макс. 10MB)
    - Попробуйте скачать шаблон и использовать его
  </Accordion>
  
  <Accordion title="Столбцы не обнаружены">
    - Убедитесь, что первая строка содержит заголовки
    - Проверьте, что названия столбцов соответствуют ожидаемым
    - Вручную сопоставьте столбцы, если нужно
    - Используйте шаблонный файл для справки
  </Accordion>
  
  <Accordion title="План этажа не загружается">
    - Проверьте формат файла (PNG, JPG, SVG)
    - Проверьте размер файла (макс. 5MB на изображение)
    - Убедитесь, что изображение не повреждено
    - Попробуйте другой формат изображения
  </Accordion>
</AccordionGroup>

## Лучшие практики

<Tip>
**Используйте шаблоны**: Всегда начинайте с предоставленного шаблона, чтобы обеспечить правильное форматирование и избежать ошибок.
</Tip>

<Tip>
**Регулярные резервные копии**: Регулярно экспортируйте данные вашего проекта для резервного копирования.
</Tip>

<Tip>
**Последовательное именование**: Используйте последовательные соглашения об именовании для квартир (например, 101, 102, 201, 202).
</Tip>

<Tip>
**Высококачественные изображения**: Используйте изображения планов этажей высокого разрешения для лучшей визуализации.
</Tip>
