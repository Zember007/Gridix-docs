# Ценообразование и калькулятор рассрочки

Настройте параметры ценообразования и включите расчеты рассрочки, чтобы помочь клиентам понять варианты оплаты для ваших квартир.

## Конфигурация валюты

### Установка валюты проекта

<Steps>
  <Step title="Откройте настройки проекта">
    Перейдите в редактор проекта и откройте Настройки.
  </Step>
  
  <Step title="Выберите валюту">
    Выберите валюту для вашего проекта:
    - USD (Американский доллар)
    - EUR (Евро)
    - GEL (Грузинский лари)
    - RUB (Российский рубль)
    - И другие поддерживаемые валюты
  </Step>
  
  <Step title="Примените валюту">
    Валюта применяется ко всем ценам в проекте.
  </Step>
</Steps>

### Отображение валюты

Настройте, как отображаются цены:

- **Позиция символа**: До или после цены (€100 или 100€)
- **Десятичные знаки**: Количество десятичных знаков (0, 2 и т.д.)
- **Разделитель тысяч**: Запятая, точка или пробел
- **Формат валюты**: Опции пользовательского форматирования

## Ценообразование квартир

### Установка цен

<Steps>
  <Step title="Выберите квартиру">
    Выберите квартиру, которую хотите оценить.
  </Step>
  
  <Step title="Введите цену">
    Введите общую цену в валюте вашего проекта.
  </Step>
  
  <Step title="Установите цену за квадратный метр">
    Опционально установите цену за квадратный метр (автоматически рассчитывается или вручную).
  </Step>
  
  <Step title="Примените скидку (необязательно)">
    Установите процент скидки или фиксированную сумму, если применимо.
  </Step>
  
  <Step title="Сохраните цену">
    Цена сохраняется и отображается в виджете.
  </Step>
</Steps>

### Ценовые диапазоны

Определите ценовые категории для фильтрации:

- **Бюджетный**: Нижний ценовой диапазон
- **Стандартный**: Средний ценовой диапазон
- **Премиум**: Высокий ценовой диапазон
- **Люкс**: Самый высокий ценовой диапазон

## Калькулятор рассрочки

### Включение калькулятора рассрочки

<Steps>
  <Step title="Перейдите в настройки рассрочки">
    Перейдите в Настройки проекта → Калькулятор рассрочки.
  </Step>
  
  <Step title="Включите калькулятор">
    Переключите "Включить калькулятор рассрочки" в положение ВКЛ.
  </Step>
  
  <Step title="Настройте параметры">
    Установите параметры калькулятора (см. конфигурацию ниже).
  </Step>
  
  <Step title="Протестируйте калькулятор">
    Используйте предварительный просмотр для тестирования расчетов рассрочки.
  </Step>
</Steps>

### Конфигурация калькулятора

Настройте опции рассрочки:

<CardGroup cols={2}>
  <Card title="Первоначальный взнос">
    - Минимальный процент первоначального взноса (по умолчанию: 20%)
    - Максимальный процент первоначального взноса
    - Сумма первоначального взноса по умолчанию
  </Card>
  <Card title="Период рассрочки">
    - Минимальные месяцы (по умолчанию: 6)
    - Максимальные месяцы (по умолчанию: 24)
    - Период по умолчанию
    - Шаг приращения
  </Card>
</CardGroup>

### Процентная ставка (необязательно)

Если применимо, настройте процент:

- **Процентная ставка**: Годовой процент
- **Метод расчета**: Простой или сложный процент
- **Частота платежей**: Ежемесячно, ежеквартально и т.д.

## Опции отображения цены

### Форматирование цены

Настройте отображение цены:

```javascript
// Примеры форматов:
"$100,000"      // С символом и разделителем
"100 000 USD"   // С пробелом-разделителем
"€100.000"      // Европейский формат
"100,000.00"    // С десятичными знаками
```

### Цена за квадратный метр

Отображение цены за квадратный метр:

<Steps>
  <Step title="Включите цену за м²">
    Переключите "Показать цену за квадратный метр" в настройках отображения.
  </Step>
  
  <Step title="Настройте отображение">
    Установите формат отображения:
    - Показать рядом с общей ценой
    - Показать в деталях квартиры
    - Рассчитывать автоматически
  </Step>
</Steps>

## Скидки и акции

### Установка скидок

<Steps>
  <Step title="Выберите квартиру">
    Выберите квартиру для скидки.
  </Step>
  
  <Step title="Установите тип скидки">
    Выберите:
    - Процентная скидка (например, 10%)
    - Скидка фиксированной суммы (например, $10,000)
  </Step>
  
  <Step title="Установите период скидки">
    Настройте:
    - Дата начала
    - Дата окончания
    - Условия (если есть)
  </Step>
  
  <Step title="Отобразите скидку">
    Скидка показывается с оригинальной ценой и ценой со скидкой.
  </Step>
</Steps>

### Промо-цены

Создайте специальные ценовые предложения:

- **Скидки для ранних покупателей**: Скидки для ранних покупателей
- **Сезонные акции**: Ограниченные по времени предложения
- **Скидки за объем**: Скидки за несколько покупок
- **Реферальные скидки**: Скидки за рекомендации

## Конвертация валют

### Поддержка мультивалютности

<Note>
Gridix поддерживает отображение цен в нескольких валютах. Клиенты могут переключать валюты в виджете.
</Note>

Поддерживаемые валюты включают:
- USD, EUR, GBP
- GEL, RUB, TRY
- AED, SAR (для рынков Ближнего Востока)
- И другие

### Переключатель валют

Включите переключатель валют в виджете:

- **Валюта по умолчанию**: Установите валюту проекта
- **Доступные валюты**: Выберите, какие валюты показывать
- **Курсы валют**: Настройте курсы валют (вручную или API)
- **Автообновление**: Включите автоматическое обновление курсов

## Расчеты цен

### Автоматические расчеты

Gridix может автоматически рассчитывать:

- **Цена за квадратный метр**: Общая цена ÷ Площадь
- **Ежемесячный платеж**: Калькулятор рассрочки
- **Итого с процентами**: Если процент настроен
- **Сумма скидки**: Расчеты скидок

### Ручное переопределение

Вы можете вручную переопределить любые рассчитанные значения:
- Установить пользовательскую цену за м²
- Переопределить расчеты рассрочки
- Установить пользовательские суммы скидок

    - Excel с формулами
    - CSV для импорта
    - JSON для API
  </Step>
  
  <Step title="Выберите данные">
    Выберите, что экспортировать:
    - Цены квартир
    - Графики рассрочки
    - История цен
    - Скидки
  </Step>

  </Step>
</Steps>

## Устранение неполадок

<AccordionGroup>
  <Accordion title="Цена отображается неправильно">
    - Проверьте настройки валюты
    - Проверьте конфигурацию формата цены
    - Убедитесь, что валюта установлена для проекта
    - Очистите кеш браузера
  </Accordion>
  
  <Accordion title="Калькулятор рассрочки не работает">
    - Проверьте, что калькулятор включен
    - Проверьте настройки конфигурации
    - Убедитесь, что установлена действительная цена
    - Протестируйте с разными значениями
  </Accordion>
  
  <Accordion title="Конвертация валют неправильная">
    - Проверьте, что курсы валют актуальны
    - Проверьте настройки конвертации валют
    - Обновите курсы валют вручную при необходимости
    - Свяжитесь с поддержкой по вопросам API
  </Accordion>
</AccordionGroup>
