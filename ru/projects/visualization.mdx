# Настройки визуализации

Настройте внешний вид и поведение планов этажей и квартир вашего проекта. Сделайте ваши проекты визуально привлекательными и удобными для навигации клиентам.

## Доступ к настройкам визуализации

Перейдите в редактор проекта и откройте раздел "Визуализация" или "Настройки отображения". Здесь вы можете настроить все визуальные аспекты вашего проекта.

## Настройка темы

### Цвета проекта

<Steps>
  <Step title="Выберите цвет темы">
    Выберите основной цвет для вашего проекта:
    - Используйте цветовую палитру
    - Введите hex код цвета
    - Выберите из предустановок
  </Step>
  
  <Step title="Примените к элементам">
    Цвет темы применяется к:
    - Активным состояниям
    - Выделениям выбора
    - Кнопкам и призывам к действию
    - Акцентам бренда
  </Step>
  
  <Step title="Предварительный просмотр изменений">
    Используйте режим предварительного просмотра, чтобы увидеть, как выглядят цвета перед сохранением.
  </Step>
</Steps>

### Настройки фона

Настройте внешний вид фона:

- **Сплошной цвет**: Выберите сплошной цвет фона
- **Градиент**: Создайте градиентные фоны
- **Изображение**: Загрузите пользовательское изображение фона
- **Прозрачный**: Прозрачный фон для встраивания

## Отображение плана этажа

### Настройки масштабирования

<Steps>
  <Step title="Установите масштаб по умолчанию">
    Настройте начальный уровень масштабирования:
    - Подогнать к экрану (по умолчанию)
    - Конкретный процент масштабирования
    - Масштабирование к конкретной области
  </Step>
  
  <Step title="Лимиты масштабирования">
    Установите минимальный и максимальный уровни масштабирования:
    - Мин. масштаб: Предотвратить слишком сильное уменьшение
    - Макс. масштаб: Ограничить максимальное увеличение
  </Step>
  
  <Step title="Элементы управления масштабированием">
    Включите/выключите элементы управления масштабированием:
    - Кнопки увеличения/уменьшения
    - Масштабирование колесом мыши
    - Масштабирование жестом (мобильное)
  </Step>
</Steps>

### Настройки панорамирования

Настройте поведение панорамирования:

- **Границы панорамирования**: Ограничьте панорамирование областью плана этажа
- **Плавное панорамирование**: Включите плавные анимации
- **Ограничения панорамирования**: Предотвратить панорамирование за пределами границ
- **Начальная позиция**: Установите начальную позицию панорамирования

### Сетка и наложения

- **Отображение сетки**: Показать/скрыть наложение сетки
- **Цвет сетки**: Настройте цвет линий сетки
- **Прозрачность сетки**: Настройте прозрачность сетки
- **Индикатор масштаба**: Отобразить информацию о масштабе
- **Отображение координат**: Показать координаты при наведении

## Отображение квартир

### Маркеры квартир

Настройте, как квартиры отображаются на планах этажей:

<CardGroup cols={2}>
  <Card title="Стиль маркера">
    - Форма (круг, квадрат, многоугольник)
    - Размер
    - Стиль границы
    - Паттерн заливки
  </Card>
  <Card title="Цвета статуса">
    - Доступна: Зеленый
    - Продана: Красный
    - Забронирована: Желтый
    - Скрыта: Серый
  </Card>
</CardGroup>

### Метки квартир

Настройте метки квартир:

- **Показать номера**: Отобразить номера квартир
- **Позиция метки**: Над, под или внутри маркера
- **Стиль метки**: Шрифт, размер, цвет
- **Формат метки**: Опции пользовательского форматирования

### Эффекты наведения

<Steps>
  <Step title="Включите эффекты наведения">
    Переключите эффекты наведения для квартир.
  </Step>
  
  <Step title="Настройте отображение при наведении">
    Установите, что появляется при наведении:
    - Номер квартиры
    - Основная информация (комнаты, площадь, цена)
    - Быстрый предварительный просмотр
    - Индикатор статуса
  </Step>
  
  <Step title="Анимация наведения">
    Выберите стиль анимации:
    - Появление
    - Скольжение вверх
    - Масштабирование
    - Нет
  </Step>
</Steps>

## Интерактивные функции

### Поведение при клике

Настройте, что происходит, когда клиенты кликают на квартиры:

<Tabs>
  <Tab title="Показать детали">
    Отобразить детали квартиры в:
    - Модальном окне
    - Боковой панели
    - Новой странице
    - Развернуть встроенно
  </Tab>
  
  <Tab title="Действия">
    Запустить действия:
    - Открыть форму контакта
    - Показать форму запроса
    - Перейти на страницу деталей
    - Призыв к действию
  </Tab>
</Tabs>

### Функция поиска

<Steps>
  <Step title="Включите поиск">
    Переключите функцию поиска квартир.
  </Step>
  
  <Step title="Настройте опции поиска">
    Установите поля для поиска:
    - Номер квартиры
    - Этаж
    - Комнаты
    - Ценовой диапазон
    - Статус
  </Step>
  
  <Step title="Отображение результатов поиска">
    Настройте отображение результатов поиска:
    - Выделить соответствующие квартиры
    - Масштабировать к результатам
    - Показать список результатов
  </Step>
</Steps>

### Фильтры

Включите опции фильтрации:

- **Фильтр по статусу**: Фильтровать по доступности
- **Фильтр по цене**: Фильтровать по ценовому диапазону
- **Фильтр по комнатам**: Фильтровать по количеству комнат
- **Фильтр по площади**: Фильтровать по диапазону площади
- **Пользовательские фильтры**: Фильтровать по пользовательским полям

## Оптимизация для мобильных устройств

### Адаптивный дизайн

<Check>
Gridix автоматически оптимизирует для мобильных устройств:
- Элементы управления, удобные для касания
- Адаптивная компоновка
- Оптимизированные для мобильных изображения
- Жесты смахивания
</Check>

### Настройки для мобильных устройств

- **Элементы управления касанием**: Включить/выключить жесты касания
- **Мобильная компоновка**: Оптимизировать для маленьких экранов
- **Загрузка изображений**: Ленивая загрузка изображений на мобильных
- **Производительность**: Оптимизировать для мобильной производительности

## Настройки анимации

### Переходы

Настройте анимации переходов:

- **Переходы между страницами**: Между видами
- **Анимации квартир**: При выборе квартир
- **Анимации масштабирования**: Плавные переходы масштабирования
- **Анимации панорамирования**: Плавное панорамирование

### Производительность

<Warning>
Избыточные анимации могут повлиять на производительность. Используйте анимации разумно для лучшего пользовательского опыта.
</Warning>

Оптимизируйте производительность:
- Уменьшите продолжительность анимации
- Ограничьте одновременные анимации
- Используйте аппаратное ускорение
- Оптимизируйте размеры изображений

## Режим предварительного просмотра

### Тестирование изменений

<Steps>
  <Step title="Включите предварительный просмотр">
    Нажмите кнопку "Предварительный просмотр" в редакторе проекта.
  </Step>
  
  <Step title="Протестируйте визуализацию">
    Протестируйте все визуальные настройки:
    - Цвета и темы
    - Масштабирование и панорамирование
    - Взаимодействия с квартирами
    - Мобильный вид
  </Step>
  
  <Step title="Внесите корректировки">
    Вернитесь в редактор для внесения изменений при необходимости.
  </Step>
</Steps>

## Настройки экспорта

### Экспорт изображений

Настройте настройки экспорта:

- **Разрешение**: Разрешение экспорта (HD, Full HD, 4K)
- **Формат**: PNG, JPG, SVG
- **Фон**: Включить/исключить фон
- **Водяной знак**: Добавить водяной знак при необходимости

### Настройки печати

Оптимизируйте для печати:

- **Размер страницы**: A4, Letter и т.д.
- **Ориентация**: Портретная или Альбомная
- **Поля**: Установите поля печати
- **Масштаб**: Подогнать к странице или конкретный масштаб

## Лучшие практики

<Tip>
**Последовательный брендинг**: Используйте цвета вашего бренда последовательно во всех визуальных элементах.
</Tip>

<Tip>
**Четкие метки**: Убедитесь, что номера квартир и метки четко видны.
</Tip>

<Tip>
**Интуитивная навигация**: Сделайте элементы управления масштабированием и панорамированием легкими для поиска и использования.
</Tip>

<Tip>
**Производительность прежде всего**: Балансируйте визуальную привлекательность с производительностью, особенно на мобильных устройствах.
</Tip>

## Устранение неполадок

<AccordionGroup>
  <Accordion title="Изменения не видны">
    - Очистите кеш браузера
    - Обновите страницу
    - Проверьте, были ли изменения сохранены
    - Проверьте, что выбран правильный проект
  </Accordion>
  
  <Accordion title="Проблемы с производительностью">
    - Уменьшите сложность анимации
    - Оптимизируйте размеры изображений
    - Отключите ненужные функции
    - Проверьте интернет-соединение
  </Accordion>
  
  <Accordion title="Проблемы с отображением на мобильных">
    - Протестируйте на реальном устройстве
    - Проверьте настройки адаптивности
    - Проверьте, что элементы управления касанием включены
    - Очистите кеш мобильного браузера
  </Accordion>
</AccordionGroup>
